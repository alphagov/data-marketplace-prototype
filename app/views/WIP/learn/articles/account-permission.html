Account permissions for the {{ serviceName }}

To search for data in the {{ serviceName }}, and submit a data share request, you only need a gov.uk email address.

If you want to add a metadata description of your data, or if you want to review a request from someone to see your data, you’ll need to request additional permissions.

What permissions you need

If you want to add a description of your data (also referred to as metadata) to the {{ serviceName }} so your data is available for colleagues in government organisations to find, you’ll need permission to add a data description.

If you want to review a request from colleagues in government organisations who want you to share your data with them, you’ll need permission to review a data share request.

What permissions enable you to do

Add a data description permission will enable you to add a metadata description of your data to the Data Marketplace so it can be found by other colleagues across government.

Review a data share request permission will enable you to review a data share request from colleagues who have found a description of your data and would like you to share it with them.

How to request permission

In the {{ serviceName }} help and support form select either:

  • Permission to add data descriptions
  • Permission to review data share requests

The form will be sent to your organisation, or team, administrator for approval. They’ll send you an email to confirm whether or not you have been given permission.
