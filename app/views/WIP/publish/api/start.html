Connect to the Data Marketplace API

This API allows you to automatically publish and update listings in the Data Marketplace’s catalogue, using information from your organisation’s data catalogue or data store.

You’ll need someone with technical expertise and knowledge of APIs to do this integration. Typically this will be an API developer.

If you’re a developer, you can get started with the API.

Ask your developer to join the Data Marketplace

Your API developer must work in your organisation and have a government email address with your organisation’s domain (for example, hmrc.gov.uk).

They need to create a Data Marketplace account and then request the Metadata Publisher role from their user profile.

They will then be able to connect to the API.
